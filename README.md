-PHP Database with login-

A learning-focused PHP project demonstrating how to connect a PHP application to a MySQL database, implement basic CRUD operations (Create, Read, Update, Delete) and  access to the app with an authentication system

-Features

Database connection using PHP<br>
Read and display database records<br>
Insert new records<br>
Delete existing records<br>
Update existing records<br>
Clear file separation for each operation
Login system with password and user authentication

-Technologies

PHP<br>
MySQL<br>
HTML<br>
Bootstrap library<br>
Local development using XAMPP
